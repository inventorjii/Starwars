Star Wars API Frontend :-

Features :-

1. Provides UI to take inputs from user.
2. Supports offline and Online mode.
3. Built with Vue js
4. Deployed on Minikube.

Technologies Used :-

1. Vue version 5.0.8
2. Node version 22.14.0
3. Docker and Kubernetes

Run Locally :-

1. npm run dev
2. Build -> npm run build
